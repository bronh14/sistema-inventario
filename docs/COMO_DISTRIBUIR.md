# Cómo Distribuir el Sistema de Inventario

## 🚀 Opciones de Distribución

### 1. Instalador Ultra Simple (Recomendado)

**Archivo a distribuir:** `instalar.bat`

**Cómo usar:**
1. Copia el archivo `instalar.bat` a cualquier dispositivo Windows
2. Haz doble clic en `instalar.bat`
3. El instalador descargará automáticamente todo lo necesario desde GitHub

**Ventajas:**
- ✅ Solo un archivo de 1KB
- ✅ Se puede enviar por email, WhatsApp, etc.
- ✅ Descarga automáticamente Python, Git y el proyecto
- ✅ Instalación completamente automática
- ✅ Funciona en cualquier Windows 10/11

### 2. Instalador Completo

**Archivo a distribuir:** `instalar_sistema_inventario.bat`

**Cómo usar:**
1. Copia el archivo `instalar_sistema_inventario.bat` al dispositivo
2. Ejecuta como administrador
3. Instalación automática completa

**Ventajas:**
- ✅ Instalador completo con todas las funciones
- ✅ Manejo de errores avanzado
- ✅ Métodos alternativos de descarga
- ✅ Limpieza automática de archivos temporales

### 3. Instalación desde GitHub (Para usuarios técnicos)

**Comando a compartir:**
```bash
curl -L https://raw.githubusercontent.com/Bronh14/sistema-inventario/main/instalar.bat -o instalar.bat && instalar.bat
```

## 📋 Instrucciones para el Usuario Final

### Opción 1: Instalador Simple
1. **Descarga** el archivo `instalar.bat`
2. **Guárdalo** en tu escritorio o cualquier carpeta
3. **Haz doble clic** en `instalar.bat`
4. **Espera** a que termine la instalación
5. **¡Listo!** El sistema se ejecutará automáticamente

### Opción 2: Instalador Completo
1. **Descarga** el archivo `instalar_sistema_inventario.bat`
2. **Haz clic derecho** → "Ejecutar como administrador"
3. **Sigue** las instrucciones en pantalla
4. **¡Listo!** El sistema se instalará completamente

## 🔧 Requisitos del Sistema

- **Windows 10 o 11**
- **Conexión a internet** (solo para la instalación)
- **Permisos de administrador** (recomendado)

## 📦 Lo que se Instala Automáticamente

- ✅ **Python 3.11** (si no está instalado)
- ✅ **Git** (si no está instalado)
- ✅ **Todas las dependencias** del proyecto
- ✅ **El sistema completo** desde GitHub
- ✅ **Acceso directo** en el escritorio
- ✅ **Base de datos** inicial

## 🎯 Casos de Uso

### Para Empresas
1. Envía `instalar.bat` por email a todos los empleados
2. Cada uno ejecuta el archivo en su computadora
3. El sistema se instala automáticamente en todas las máquinas

### Para Clientes
1. Adjunta `instalar.bat` en un email
2. El cliente descarga y ejecuta el archivo
3. Instalación automática sin complicaciones

### Para Distribución Masiva
1. Sube `instalar.bat` a Google Drive, Dropbox, etc.
2. Comparte el enlace
3. Cualquiera puede descargar e instalar

## 🔄 Actualizaciones

### Para el Desarrollador
1. Sube los cambios a GitHub
2. Los usuarios ejecutan `instalar.bat` nuevamente
3. Se descarga automáticamente la versión más reciente

### Para el Usuario
1. Ejecuta `instalar.bat` nuevamente
2. Se actualiza automáticamente a la última versión
3. No se pierden los datos de la base de datos

## 🛠️ Solución de Problemas

### Error: "No se pudo descargar"
- Verifica la conexión a internet
- Ejecuta como administrador
- Intenta nuevamente

### Error: "Permisos denegados"
- Haz clic derecho → "Ejecutar como administrador"
- Desactiva temporalmente el antivirus

### Error: "Python no se instaló"
- El instalador intentará métodos alternativos
- Si falla, instala Python manualmente desde python.org
